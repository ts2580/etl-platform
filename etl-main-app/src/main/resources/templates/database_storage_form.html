ℹ️ 기본 정보

저장소 식별 정보

운영에서 알아보기 쉬운 이름과 설명, 암호화 키를 먼저 입력해 주세요.

이 키는 서버에 저장하지 않고, 자격증명 암호화/복호화 시점에만 사용해요.

ℹ️ 연결 설정

DB 접속 정보

DB 종류는 콤보박스로 직접 선택하고, URL은 그대로 입력하는 방식입니다.

Oracle은 사용자명 기준 스키마를 사용해요. 사용자명을 바꾸면 스키마도 함께 맞춰져요.

PostgreSQL 라우팅은 etl_sfdc DB 안에 org_ 접두사 스키마를 자동 생성해서 사용해요.
