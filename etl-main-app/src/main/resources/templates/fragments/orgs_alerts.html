토큰 갱신 실패

토큰 갱신에 실패했어요. 접근 토큰 만료일 수 있어요. 로그인 재인증이 필요할 수 있습니다.

⚠️

CID/CPW는 수정됐지만 토큰 재갱신 실패

새 인증정보로 access token이 발급되지 않았어요. 토큰/라우팅 재시도가 필요합니다.

⚠️

CID/CPW 수정 완료, 라우팅 갱신 실패

세션 토큰은 갱신됐지만 라우팅 모듈 자격 증명 갱신이 실패했습니다.

🔑

CID/CPW 변경 및 토큰 갱신 완료

선택한 Org의 Client ID / Client Secret이 갱신되었고, 세션/라우팅 토큰까지 재설정했어요.

⚠️

입력값을 먼저 확인해 주세요

Org 이름, myDomain, Client ID, Client Secret을 모두 채워야 인증을 시작할 수 있어요.

⚠️

Org 이름 형식이 맞지 않아요

Org 이름은 영문으로 시작하고, 영문/숫자/공백/_ 만 사용할 수 있어요. 길이는 2~48자예요.

⚠️

OAuth 시작에 실패했어요

입력값이나 OAuth 설정을 다시 확인한 뒤 한 번 더 시도해 주세요.

저장된 Client ID/Client Secret이 없어 재로그인 흐름을 시작하지 못했어요.

⚠️

OAuth 인증 완료 처리에 실패했어요

Salesforce 인증은 끝났더라도 앱 등록 처리 중 오류가 났을 수 있어요. 다시 시도해 주세요.

✅

새 org가 등록됐어요

Client Credentials 설정 저장 후 access token까지 자동 발급했어요.

⚠️

org는 등록됐지만 토큰 발급은 실패했어요

Client Credentials 설정은 저장됐어요. 아래 목록에서 토큰 갱신을 다시 시도하거나, 로그의 reason/body를 확인해 주세요.

🌿

기본 org가 변경됐어요

이제 해당 org를 기본 대상으로 사용해요.

🗑️

org가 삭제됐어요
